How to use a command line for EmailTaskProject.

1. cd to folder with emailtask - dobromir matuszak/emailtask

2.Available command to rise by terminal

    1. -h or --help                                         # help
    2. -ir or -incorrect-email                              # show incorrect email
    3. -s (argument) or -search (arg)                       # after -s add argument to search by him in all email
    4. -gbd or --group-by-domain                            # group by domain
    5. -feil (argument) or --find-emails-not-in-logs (arg)  # after -feil add argument which is path, email-sent.logs
